# MathViz Architect 📐

**Приложение для развития визуального мышления на уроках математики**

Интерактивное desktop-приложение для создания математических визуализаций, диаграмм и обучающих материалов для учеников 5-11 классов.

## 🎯 Возможности

### 📊 Инструменты рисования
- **Базовые фигуры**: прямоугольники, круги, треугольники
- **Полигоны**: 13 дополнительных геометрических фигур
  - Треугольники (7 типов): произвольный, равносторонний, равнобедренный, разносторонний, остроугольный, прямоугольный, тупоугольный
  - Четырёхугольники (6 типов): произвольный, параллелограмм, прямоугольник, квадрат, ромб, трапеция равнобедренная, трапеция прямоугольная
- **Линия**: рисование прямых линий drag-to-draw с опциональными стрелками на концах
- **Ластик**: удаление объектов кликом или проведением мыши
- **Текст**: редактирование по двойному клику
- **Диаграммы**: столбчатые и круговые диаграммы с редактированием данных

### 🎯 Выделение и редактирование
- **Marquee selection**: выделение нескольких объектов областью (drag на пустом месте)
- **Multi-select**: Ctrl+клик для добавления/снятия объектов из выделения
- **Multi-drag**: перетаскивание всех выбранных объектов одновременно
- **Backspace в тексте**: корректное удаление символов при редактировании текста

### 🎓 Интерактивные модули (18 модулей)

Все модули имеют профессиональный двухколоночный layout с графиком слева и панелью управления справа, встроенные инструкции и tooltips для элементов управления.

#### Функции (7 модулей)
- **Линейная функция** (y = kx + b) - интерактивное изменение параметров
- **Квадратичная функция** (y = ax² + bx + c) - визуализация параболы с вершиной, плавная цикличная анимация
- **Показательная функция** (y = a·bˣ + c) - экспоненциальный рост и убывание
- **Логарифмическая функция** (y = a·log_b(x) + c) - обратная к показательной
- **Производная** - исследование функции y = ax³ + bx² + cx + d, экстремумы, точки перегиба
- **Пересечения функций** - нахождение точек пересечения двух графиков
- **Преобразования функций** - визуализация сдвигов, растяжений и отражений графиков (y = k·f(d·x - a) + c)
- **Исследование функции** - полный анализ кубической функции: критические точки, экстремумы, точки перегиба, монотонность, выпуклость

#### Геометрия (5 модулей)
- **Теорема Пифагора** - анимированное доказательство с квадратами, пошаговые объяснения, динамическая формула
- **Векторы** - операции с векторами (сложение, вычитание, скалярное произведение, угол)
- **Координатная плоскость** - добавление точек кликом, изучение координат
- **Параллельные прямые** - углы при пересечении параллельных прямых секущей, корректная визуализация углов
- **Подобие треугольников** - свойства подобных треугольников и коэффициент подобия

#### Тригонометрия (3 модуля)
- **Тригонометрическая окружность** - sin, cos, tg с анимацией, переключение градусы/радианы
- **Тригонометрические функции** - графики sin, cos, tan, cot с изменением амплитуды, периода и фазы, корректная обработка асимптот
- **Окружность и график** - анимированная визуализация связи единичной окружности и синусоиды/косинусоиды в реальном времени

#### Алгебра (3 модуля)
- **Вероятностное дерево** - визуализация условных вероятностей для 2-3 событий
  - Два режима: простое дерево и условные вероятности
  - Переименование событий
  - Динамическое добавление третьего уровня
  - Автоматический расчёт итоговых вероятностей
- **Системы неравенств** - визуализация решений систем линейных неравенств

### 🎮 Режим задач
- **Категории**: Алгебра (5 класс), Геометрия, Тригонометрия
- **Иерархическая навигация**: Категория → Тема → Задачи
- **Отслеживание прогресса**: автоматическое сохранение выполненных задач
- **Типы задач**: 
  - Сравнение чисел
  - Числовые закономерности
  - Магический квадрат 3×3
  - Периметр прямоугольника
  - Определение типа треугольника

### 🎨 Возможности редактирования
- Перетаскивание объектов
- Масштабирование и вращение
- Изменение цветов и стилей
- Редактирование данных диаграмм (добавление/удаление секторов и столбцов)
- Группировка объектов
- Отмена/повтор действий (Ctrl+Z / Ctrl+Y, Ctrl+Shift+Z)
- Удаление выбранных объектов (Delete/Backspace)
- Сетка для точного позиционирования

## 🚀 Быстрый старт

### Требования
- **Node.js** 18+ ([скачать](https://nodejs.org/))
- **pnpm** (рекомендуется) или npm

### Установка pnpm (если не установлен)
```bash
npm install -g pnpm
```

### Установка зависимостей
```bash
cd mathviz-architect
pnpm install
```

### Запуск веб-версии (для разработки)
```bash
pnpm run dev
```
Откройте http://localhost:5173 в браузере

### Запуск desktop-версии (Electron)
```bash
pnpm run dev:electron
```

### Сборка production-версии
```bash
# Веб-версия
pnpm run build

# Desktop-приложение (Windows/Mac/Linux)
pnpm run build:electron
```

Готовое приложение будет в папке `release/`

Математические задачи с проверкой ответов для 5-11 классов

## 🔧 Разработка

### Добавление нового объекта Canvas

1. **Добавьте тип в `src/lib/types.ts`**:
```typescript
export interface MyObjectType extends CanvasObject {
  type: 'myobject';
  data: {
    // специфичные поля
  };
}
```

2. **Добавьте в `ObjectCreator.tsx`**:
```typescript
{
  id: 'myobject',
  name: 'Мой объект',
  icon: MyIcon,
  createObject: (): AnyCanvasObject => ({
    id: generateId(),
    type: 'myobject',
    // ... остальные поля
  }),
}
```

3. **Добавьте рендер в `Canvas.tsx`**:
```typescript
case 'myobject': {
  // логика рендеринга
  return <g>...</g>;
}
```

### Добавление интерактивного модуля

1. Создайте компонент в `src/components/interactive/`
2. Используйте стандартный layout:
   - Внешний контейнер: `flex flex-col h-full bg-white`
   - Горизонтальный контейнер: `flex-1 flex overflow-hidden`
   - График слева: `flex-1 overflow-auto` с CoordinateGrid
   - Панель справа: `w-96 border-l overflow-y-auto`
3. Зарегистрируйте модуль в `src/modules/index.ts`:
   - Импортируйте компонент
   - Вызовите `registerModule()` с метаданными модуля

## 🌐 Деплой на GitHub Pages

Проект настроен для автоматического деплоя на GitHub Pages через GitHub Actions.

### Автоматический деплой

При каждом push в ветку `main` автоматически запускается workflow, который:
1. Устанавливает зависимости
2. Собирает production-версию
3. Деплоит на GitHub Pages

### Настройка (один раз)

1. **Включите GitHub Pages в настройках репозитория:**
   - Перейдите в Settings → Pages
   - Source: выберите "GitHub Actions"

2. **Проверьте права доступа:**
   - Settings → Actions → General → Workflow permissions
   - Выберите "Read and write permissions"

3. **После первого push в main:**
   - Workflow запустится автоматически
   - Приложение будет доступно по адресу: `https://[username].github.io/MathViz-Architect/`

### Ручной деплой

Вы можете запустить деплой вручную:
1. Перейдите в Actions → Deploy to GitHub Pages
2. Нажмите "Run workflow"
3. Выберите ветку `main` и нажмите "Run workflow"

### Локальная сборка для GitHub Pages

Чтобы протестировать сборку локально:

```bash
cd mathviz-architect
GITHUB_PAGES=true pnpm build
pnpm preview
```

## 🐛 Известные проблемы и решения

### Проблема: Веб-версия не запускается
**Решение**: 
```bash
# Удалите node_modules и переустановите
rm -rf node_modules
pnpm install
pnpm run dev
```

### Проблема: Запуск с доступом по сети
**Решение**: Используйте флаг --host
```bash
cd "C:\Users\Timur\Desktop\прокет\mathviz-architect"
pnpm run dev -- --host
```
Приложение будет доступно по адресу http://localhost:5173 и по IP-адресу вашего компьютера

### Проблема: Ошибки TypeScript
**Решение**: Проверьте версию TypeScript
```bash
pnpm list typescript
# Должна быть ~5.6.2
```

### Проблема: Electron не запускается
**Решение**: Убедитесь, что веб-версия собрана
```bash
pnpm run build
pnpm run electron
```
